# PatchPilot

PatchPilot turns a messy pull request into a five-minute review route. It
groups related files, highlights risky changes, and gives reviewers a compact
map before they open the diff.

## Features

- Groups changed files by behavior instead of folder
- Flags migrations, auth boundaries, and dependency changes
- Generates a shareable review route with no hosted service
- Keeps source code on the developer machine

## Quick start

```bash
npm install
npm run dev
```
